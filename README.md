# Education Research Opportunity Scanner

A self-contained browser app for ranking education research opportunities. It accepts pasted opportunity text or simple CSV rows, scores each lead against a configurable research profile, flags urgent deadlines, and exports the filtered shortlist as CSV.

Open `index.html` in a browser to use it. No install step is required.

CSV format:

```csv
Title, Source, Deadline, Description
```

The scanner works best when each row includes a title, funder or source, deadline, and a short description.
